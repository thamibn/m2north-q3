# ****How to install and run

##### ****Step 1

``` git clone https://github.com/thamibn/m2north-q3.git ```
##### ****Step 2
copy ```.env.example``` to ``` .env```
##### ****Step 3
``` composer install ```
##### ****Step 4
``` npm install ```
##### ****Step 5
``` npm run dev ```
##### ****Step 6
``` php artisan key:generate ```
##### ****Step 7
setup your database credentials in ``` .env ``` file

run ``` php artisan migrate ```
##### ****Final Step

``` php artisan serve ```
